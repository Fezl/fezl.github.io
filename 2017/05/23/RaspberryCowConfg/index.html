准备工作

  1. 已经能正常进入Raspbian系统的树莓派
  2. ss账号

其实cow的 README.md 文档已经说的十分详细了。但还是走了一点弯路，毕竟第一次设置这种类型的代理，还没有相关的概念。

安装

在树莓派终端中输入指令后回车，会询问要讲cow安装到哪个地方，这里注意需要绝对路径，当然你也可以直接回车默认安装在当前路径。

配置

安装完成后会在路径下看到一个名为 cow 的可执行文件。配置文件自动生成在 ~/.cow/rc 路径下。

1. listen地址

这里的地址填树莓派在局域网内的ip地址。查看ip地址命令如下：

例：listen = http://192.168.1.100:1088

2. shadowsocks账号配置

这里按照规则，填写你的ss账号信息，注意密码、服务器IP、端口、加密类型统统不需要加引号，直接写。
例：proxy = ss://aes-128-cfb:12345678@123.123.123.123:8388
